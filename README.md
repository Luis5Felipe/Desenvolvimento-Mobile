### Possíveis questões de prova

- Caracteristicas dos Métodos Node.js
  1. Todo método espera a ocorrência de um evento
     - O evento esperado pelo método createserver() é a requisisão feita pelo cliente.
  2. Todo médoto possui uma função anônima
     - toda função anônima possui parâmentros.
     - Os parêmetros de função anônima de método creatserver() são req(resquest) e res(response)
  3. A função anônima é executada sempre que o evento esperado pelo método ocorre.
